# Go CI/CD Project

Простой веб-сервер на Go с настроенным CI/CD pipeline в GitHub Actions.

## Функциональность
- REST API эндпоинт `/health` для проверки статуса
- Юнит-тесты для всех компонентов
- Автоматическая сборка и тестирование на Linux, Windows, macOS
- Автоматический деплой на удаленный сервер

## Локальный запуск
go run ./cmd/server

## Запуск тестов
go test -v ./...
